Programming 2 Final Project - FGCU

Project Overview This repository contains the final project for the Programming 2 course at Florida Gulf Coast University. The application was developed as a collaborative effort between two students.

The project is a desktop productivity suite built using C++ and the Qt6 framework. It features a multi-window graphical user interface that requires user authentication to access a suite of integrated tools.

Features

User Authentication: A secure login and account registration system.

Central Dashboard: A main navigation window to launch various utilities.

Digital Clock: A real-time clock application for time management.

Notepad: A built-in text editor for creating and saving notes.

Calculator: A functional tool for basic arithmetic operations.

Technical Details

Language: C++

Framework: Qt6 (Widgets, Core, Gui)

Build System: CMake

Platform: Developed Primarily for Windows 10, features some visible bugs in Windows 11.

Installation and Usage To run this project, the Qt SDK must be installed.

Clone the repository to your local machine.

Open the CMakeLists.txt file in Qt Creator or your preferred C++ IDE.

Configure the project using a Qt 6.x kit.

Build and run the application.
